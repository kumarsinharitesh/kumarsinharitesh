<details>
<summary><b>⬡ InferPrompt — Multi-LLM Inference Playground</b></summary>

<br/>

A live AI inference platform designed for engineers who need to evaluate, compare, and benchmark LLM outputs with production-grade observability. Ships real-time token-by-token streaming with per-token latency visualization across four major providers.

| Attribute | Detail |
|-----------|--------|
| **Stack** | Next.js, TypeScript, Sarvam AI, Gemini, Groq, OpenAI Streaming APIs |
| **Core Engine** | ABTD (A/B Token Diff) — proprietary side-by-side model output diff algorithm |
| **Providers** | 4+ LLM providers supported simultaneously |
| **Performance** | Estimated 70% reduction in prompt evaluation effort via parallel streaming comparison |
| **Deployment** | Live on Vercel |
| **Repository** | [GitHub](https://github.com/kumarsinharitesh/InferPrompt) · [Live Demo](https://inferprompt.vercel.app) |

**Engineering Notes:** The ABTD engine computes token-level diffs across provider responses in real time, enabling engineers to identify divergence points in model reasoning without manual comparison. The streaming architecture uses SSE with backpressure-aware buffering to maintain sub-100ms time-to-first-token at the UI layer.

<br/>

</details>
